toggle(forceopt)

Toggles the item in the tree and hides/unhides this `Segment` from the Peaks waveform
Parameters:
Name Type Attributes Default Description
force boolean <optional>
 null If given, forces the item to toggle on/off. If true, force checks the checkbox, turns on the buttons, and unhides the segment in Peaks. If false, force unchecks the checkbox, turns off the buttons, and hides the segment in Peaks. If force equals this.checked, no toggling is done.
