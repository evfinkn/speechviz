# add(disposable, …disposables)

Adds Disposables to this disposer. This includes adding disposable properties of disposable and its children (if any). Items are only added if they have a dispose() method and / or can be removed from a parent.

Parameters:
Name Type Attributes Description
disposable Disposable | Array.<Disposable>

The Disposable item to add to this disposer. If disposable is an array of Disposables, each one is added.
disposables Disposable <repeatable>

Extra Disposables to add to this disposer. This allows passing in multiple items instead of having to call add() on each one individually.

# dispose()

Disposes of the items in this disposer. Calls dispose() on the items that define it and removes items from their parent if they have one.
